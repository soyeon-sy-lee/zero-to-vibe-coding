코딩을 전혀 모르는 상태라면 Codex 데스크톱 앱으로 시작하는 방식이 가장 쉽습니다. 터미널 명령어를 먼저 외우지 않아도 되고, 작업 폴더를 하나 만든 뒤 그 폴더를 Codex에 연결하면 됩니다.

Codex는 선택한 프로젝트 폴더 안에서 실제 Python 파일을 만들고, 수정하고, 실행까지 도와주는 코딩 에이전트입니다.

폴더는 프로젝트를 담는 상자입니다.

처음에는 작은 연습용 폴더 하나만 만들어서 그 안에서 작업하면 충분합니다.

내 문서
└── my-first-python
    ├── main.py
    └── README.md
파일은 실제 내용이 들어 있는 문서입니다.

main.py처럼 이름이 .py로 끝나는 파일이 Python 코드 파일입니다.

Codex는 채팅창 밖의 파일도 다룰 수 있습니다.

Codex에 “파일을 만들어줘”라고 하면 코드만 보여주는 것이 아니라, 연결한 프로젝트 폴더 안에 실제 파일을 만들 수 있습니다.

이 튜토리얼은 문법을 오래 외우기보다, 작은 프로젝트 폴더를 만들고 실제 파일을 실행해보는 흐름으로 진행됩니다.

각 단계에서는 지금 입력한 명령이 어떤 의미인지 확인하고, 마지막에는 Codex에게 직접 수정을 요청합니다.

  1. 준비하기Python과 Codex를 설치하고 터미널을 엽니다.
  2. 폴더와 파일 이해하기cd, mkdir, pwd, touch, ls로 작업 공간을 만듭니다.
  3. 첫 Python 실행하기print()로 문장을 출력하고 input()으로 사용자의 입력을 받아봅니다.
  4. Codex와 협업하기프로젝트 폴더를 열고 자연어로 코드 수정을 요청합니다.
  5. 작은 프로젝트 만들기숫자 맞추기 게임을 만들며 원하는 기능을 말로 요청하는 연습을 합니다.
